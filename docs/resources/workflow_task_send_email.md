# Resource (rootly_workflow_task_send_email)

Manages workflow send_email task.

## Example Usage

  trigger_params {
    triggers                  = ["incident_created"]
    wait                      = "30 seconds"
    incident_statuses         = ["started"]
    incident_condition_status = "IS"
  }
  enabled = true
}
